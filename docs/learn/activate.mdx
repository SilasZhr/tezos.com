##如何开始

### 1.检查您的贡献

单击下面的“检查您的贡献”按钮，以检查给定公钥哈希的建议分配。 捐款人可以在基金会筹款活动期间创建的PDF文档中找到其公共密钥哈希。
<Button 
    text="检查您的贡献"
    href="https://check.tezos.com/"
/>

### 2.获取您的激活码

需要激活码才能访问建议的相应公钥哈希分配。 为了获得这些代码，贡献者必须完成基金会的“了解您的客户/反洗钱”（“ KYC / AML”）流程。 尚未完成此过程的贡献者现在可以这样做。

<Button 
    text="开始 KYC/AML"
    href="https://verification.tezos.com/"
/> <Button
    text="KYC/AML FAQ"
    href="https://tezos.foundation/privacy-notice/"
    inverse />


### 3.激活您的建议分配

必须先激活筹款人的建议拨款，然后才能使用这些拨款。 贡献者可以通过使用激活工具来激活给定公钥哈希的建议分配。 该工具还可以揭示给定公共密钥哈希的盲密钥。 基金会提出的创世块中出现的密钥是盲密钥，而不是公共密钥哈希，可以在[此处]找到(https://tezos.foundation/wp-content/uploads/2020/04/5223213-genesis.txt)。

建议分配的公共密钥哈希余额只有在激活后才出现在Tezos块浏览器中。 贡献者激活其建议的分配后，可以查看“存储和使用”页面，以获取有关存储和使用其Tezos令牌（“ tez”）的信息。
<Button 
    text="激活"
    href="https://activate.tezos.com/"
/> <Button
    text="存储与使用"
    href="store-and-use/"
    inverse />
